# Markdown 语法速查手册

## 标题

使用 `#` 号标记标题，从 1 到 6 级：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本格式

- **粗体**: `**粗体文字**`
- *斜体*: `*斜体文字*`
- ~~删除线~~: `~~删除线~~`
- `行内代码`: `` `代码` ``

## 列表

### 无序列表
- 项目一
- 项目二
  - 子项目

### 有序列表
1. 第一步
2. 第二步
3. 第三步

## 链接和图片

- 链接: `[显示文字](URL)`
- 图片: `![替代文字](图片URL)`

## 引用

> 这是一段引用文字。
> 可以有多行。

## 代码块

```javascript
function hello() {
    console.log("Hello, World!");
}
```

## 表格

| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|

| 数据 | 数据 | 数据 |
