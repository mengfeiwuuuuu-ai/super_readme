# 程序员的效率工具推荐

作为一名程序员，好的工具可以极大地提升我们的开发效率。以下是我日常使用的一些优秀工具推荐。

## 🖥️ 编辑器 & IDE

### Visual Studio Code
VS Code 几乎是现代开发者的标配。推荐安装的扩展：

- **GitHub Copilot** - AI 编程助手
- **GitLens** - Git 增强
- **Prettier** - 代码格式化
- **Thunder Client** - API 测试

### JetBrains 系列
对于特定语言的深度开发，JetBrains IDE 提供了更专业的体验：
- PyCharm (Python)
- WebStorm (JavaScript)
- IntelliJ IDEA (Java)

## 📝 笔记 & 文档

| 工具 | 特点 | 推荐指数 |
|------|------|---------|
| Obsidian | 本地 Markdown | ⭐⭐⭐⭐⭐ |
| Notion | 全能协作 | ⭐⭐⭐⭐ |

## 🔧 终端工具

```bash
# Oh My Zsh - 终端美化
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# fzf - 模糊搜索
brew install fzf

# bat - 高亮 cat
brew install bat

# exa - 现代 ls
brew install exa
```

## 💡 小贴士

> 工具只是手段，真正的效率来自于对工作流的持续优化和对技术的深入理解。

选择适合自己的工具组合，建立高效的工作流程，才是提升效率的关键。
